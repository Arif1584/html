Q-1 Are the HTML tags and elements the same thing?

Ans: The HTML tags and elements are Not the same thing,
HTML tags are special keywords surrounded by a pair of angle brackets . Most tags come in pairs, with an opening and closing tag.
Where HTML element is a building block of an HTML document. It is a piece of code that defines the structure and content of a web page.

Q-2 What are tags and attributes in HTML?

Ans: HTML tags - HTML tags are like keywords that define how a web browser will format and display content. They contain three main parts: opening tag, content, and closing tag.
HTML attributes are used to modify the behavior or appearance of an element.
Ex: < b > b tag < /b >

HTML Attributes- HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of a HTML tags, Attributes provide additional information about element.
Ex: < a href ="https://www.google.com" > The href attribute specifies the URL of the page the link goes to.

Q-16 How are active links different from normal links?

Ans: Active links are typically links that are currently functional and lead to live web pages, while normal links can refer to any link, whether functional or broken. Active links are actively used and accessible, while normal links may or may not be functional at a given time.

Q-18 What is SVG?

Ans: SVG stands for Scalable Vector Graphics. SVG is used to define vector-based graphics for the Web. SVG defines the graphics in XML format. Every element and every attribute in SVG files can be animated.

Q-19 What is difference between HTML and XHTML?

Ans: HTML (HypertextMarkup Language) and XHTML (ExtensibleHypertext Markup Language) are both markup languages used for creating and displaying web pages. The main difference between them is the syntax and structure; HTML is more lenient in its syntax, while XHTML has a more strict syntax and follows XML rules.
